# Sources Checked

These are the main public sources used to build this starter package.

- BNB Chain hackathon overview: https://www.bnbchain.org/en/blog/build-and-compete-for-36-000-in-bnb-hack-ai-trading-agents-by-bnb-chain-coinmarketcap-and-trust-wallet
- CoinMarketCap hackathon page: https://coinmarketcap.com/api/hackathon/
- CoinMarketCap Agent Hub docs: https://coinmarketcap.com/api/documentation/ai-agent-hub
- CoinMarketCap Skills overview: https://coinmarketcap.com/api/documentation/ai-agent-hub/skills/overview
- CoinMarketCap MCP docs: https://pro.coinmarketcap.com/api/documentation/ai-agent-hub/skills/cmc-mcp
- CoinMarketCap pricing: https://coinmarketcap.com/api/pricing/

Important facts captured:

- Track 2 is Strategy Skills.
- Track 2 requires a backtestable strategy spec, not a live trading agent.
- Track 2 prize pool is $6,000: $3,000 / $2,000 / $1,000.
- Judging is based on technical execution, originality, real-world relevance, and demo quality.
- CMC Skills can sit on top of MCP, x402, CLI, or direct REST API integrations.
- CMC Basic is free and useful for current data/testing, but serious historical backtesting should use another free candle source unless paid historical CMC access is available.

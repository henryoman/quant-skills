# Reporting Standards

Use [report-design.md](report-design.md) as the canonical entry point.

- `report-design-liq.md` is the preserved general editorial source system.
- `report-design-ml.md` is the preserved quantitative evidence source system.
- `report-design-dlmm.md` is a domain companion for DLMM, execution, and
  strategy-specific reports.

The unified standard resolves conflicts between the source systems through an
explicit report profile. Research validity, uncertainty, and promotion status
come from the active framework; visual polish cannot upgrade weak evidence.

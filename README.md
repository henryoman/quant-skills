# Quant Skills

This workspace is organized into three top-level buckets:

- `cmc-skills/`: official native CoinMarketCap skills and BNB project foundation material.
- `third-party-skills/`: imported skills from outside vendors, community repos, mirrors, or other external authors.
- `alpha-gen/`: skills and research packages authored in this workspace for alpha generation, strategy research, and contest work.

Keep new folders inside one of these buckets. Do not add standalone skill folders at the repository root.
